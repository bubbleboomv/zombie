Periodiski tiek ģenerēti collectible objekti uz platformas

Ja zombijs saskaras ar collectible, objekts pazūd

Par savāktu objektu spēlētājs saņem punktus

Punktu skaits tiek attēlots UI teksta objektā ekrāna stūrī

Spēles sākumā tiek palaists taimeris, kas skaita laiku no spēles sākuma

Taimeris tiek attēlots UI slānī

Ja kāds zombijs nokrīt no platformas, spēle to reģistrē

Zaudējot spēle tiek apturēta (taimeris apstājas)

Parādās teksts “You lose”

Parādās poga, kas ļauj sākt spēli no sākuma

Restartējot spēli, taimeris sākas no jauna
